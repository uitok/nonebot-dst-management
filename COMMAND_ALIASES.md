# 中文命令别名对照表

## 概述

为了提升用户体验，插件为高频命令提供了中文别名。所有英文命令仍可正常使用。

---

## 房间管理

| 英文命令 | 中文别名 | 说明 |
|---------|---------|------|
| `/dst list [页码]` | `/dst 房间列表 [页码]`<br>`/dst 列表 [页码]` | 查看房间列表 |
| `/dst info <房间ID>` | `/dst 房间详情 <房间ID>`<br>`/dst 详情 <房间ID>` | 查看房间详情 |
| `/dst start <房间ID>` | - | 启动房间 🔒 |
| `/dst stop <房间ID>` | - | 关闭��间 🔒 |
| `/dst restart <房间ID>` | - | 重启房间 🔒 |

## 默认房间

| 命令 | 说明 |
|-----|------|
| `/dst 默认房间 <房间ID>` | 设置默认房间 |
| `/dst 查看默认` | 查看默认房间 |
| `/dst 清除默认` | 清除默认房间设置 |

---

## 使用示例

### 设置默认房间后

```bash
# 设置默认房间为 2
/dst 默认房间 2

# 使用默认房间（省略房间 ID）
/dst 玩家列表          # → 查看房间 2 的在线玩家
/dst 模组列表          # → 查看房间 2 的已安装模组
/dst 创建备份          # → 为房间 2 创建备份

# 临时使用其他房间
/dst 模组列表 3       # → 查看房间 3 的模组
```

### 中文命令示例

```bash
# 房间管理
/dst 房间列表          # 查看所有房间
/dst 房间详情 2        # 查看房间 2 的详情

# 玩家管理
/dst 玩家列表 2        # 查看房间 2 的在线玩家
/dst 踢出玩家 2 KU_xxx # 踢出玩家

# 模组管理
/dst 模组搜索 健康条   # 搜索模组
/dst 模组列表 2        # 查看已安装模组
/dst 添加模组 2 1 123456789  # 安装模组
/dst 检测模组 2        # 检测模组冲突

# 备份管理
/dst 备份列表 2        # 查看备份列表
/dst 创建备份 2        # 创建备份
/dst 恢复备份 2 backup_xxx.zip  # 恢复备份
```

---

## 注意事项

1. **向后兼容**：所有英文命令仍可正常使用
2. **优先级**：中文命令与英文命令功能完全相同
3. **权限标记**：🔒 标记的命令需要管理员权限
4. **默认房间**：设置默认房间后，大部分命令可省略房间 ID 参数
